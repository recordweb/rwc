# RecordWeb Concept (RWC)

**Editor's Draft**: https://recordweb.github.io/rwc/

**Published baseline (v1.0)**: https://doi.org/10.5281/zenodo.20475343

## About

This repository contains the Editor's Draft of the RecordWeb Concept (RWC), developed by the [RecordWeb W3C Community Group](https://github.com/recordweb).

RWC is the conceptual counterpart to the normative [RecordWeb Protocol (RWP)](https://recordweb.github.io/rwp/).
